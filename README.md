## Como Ejecutar

```

Este es un programa donde buscar un lugar que retorna la ciudad, el clima y que crea un archivo json como tu DB 
para almacenar tus tareas.

```

```

* Asegurate de tener nodejs/npm instalando

* Crear una cuenta en Mapbox y en OperWeather

* Colocar las AppiKey en el .env

* Instala las dependencias

$ npm install

* Para ejecturalo sigue el ejemplo:

$ npm start || node index.js

```
